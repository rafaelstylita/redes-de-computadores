# Redes de Computadores

## 📌 Descrição
Repositório contendo implementação de comunicação cliente-servidor desenvolvida na disciplina de Redes de Computadores da Universidade Federal de Ouro Preto (UFOP).

O projeto utiliza sockets em Python para estabelecer conexão entre duas máquinas distintas através de endereços IP.

---

## 🌐 Funcionamento

O sistema é composto por:

- **Servidor**: responsável por receber conexões e processar requisições
- **Cliente**: responsável por se conectar ao servidor e enviar dados

A comunicação ocorre via rede utilizando protocolos de baixo nível (sockets).

---

## 🛠️ Tecnologias utilizadas

- Python
- Sockets
